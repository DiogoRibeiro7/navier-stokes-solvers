Accelerate Your Computational Fluid Dynamics Research

Navier-Stokes Solvers delivers a battle-tested toolkit featuring second-order finite differences and Fourier spectral discretisations, designed for rapid prototyping and reproducible research.

Quick Start

git clone {{ site.github.repository_url | default: 'https://github.com/your-org/navier-stokes-solvers' }}
cd navier-stokes-solvers
make all
python scripts/plot_solution.py results/fd_final_solution.dat -o figures/fd_demo

Need a deeper dive? Follow the Your First Simulation tutorial.

High-Order Solvers

Switch between implicit finite differences and Fourier pseudospectral methods using the same API, enabling side-by-side validation.

Robust Time Integration

Leverage adaptive Backward Euler and classical RK4 schemes with automated CFL control for stability without manual tuning.

Insightful Analytics

Built-in diagnostics track energy, enstrophy, and convergence, while companion Python scripts create publication-ready figures.

Extensible Architecture

Modular C code, rich documentation, and a contributor guide make it straightforward to add new physics or numerical schemes.
